Another day I was working on a project where we had two repositories like this:

First: I will call it “Service X”:

  • RabbitMQ

First: I will call it “Magento”

  • PHP
  • Nginx
  • MySql
  • RabbitMQ

Both repositories was set up with Docker Compose. And we needed to join them in the way so that Magento communicates with Service X via RabbitMQ.
